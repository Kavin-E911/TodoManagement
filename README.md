# Todo Management System

## Project Description

The Todo Management System is a web application developed using Flask and SQLite. It allows users to create an account, log in securely, create personal todo lists, add tasks, edit tasks, delete tasks, and mark tasks as completed. Each user's data is private and cannot be accessed by other users.

---

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

---

## Features

- User Registration
- User Login & Logout
- Create Todo Lists
- Add, Edit and Delete Tasks
- Mark Tasks as Completed
- Filter Tasks by Tags
- Personal Todo Lists for Each User
- Responsive Dark Theme Interface

## How to Run

1. Clone the repository.

2. Go to the backend folder.

3. Install the required packages.

```bash
pip install -r requirements.txt
```

4. Run the application.

```bash
python app.py
```

5. Open the browser.

```
http://127.0.0.1:5000
```
